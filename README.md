# Inventory Management System 📦

A lightweight, browser-based application to manage product stocks efficiently using Vanilla JavaScript and Local Storage.

## ✨ Key Features
* **CRUD Operations:** Add and delete inventory items in real-time.
* **Data Persistence:** Uses `localStorage` to save data so it remains available even after refreshing the page.
* **Responsive UI:** Styled with CSS to work on different screen sizes.

## 🛠️ Technical Stack
* **Language:** JavaScript (ES6+)
* **Frontend:** HTML5, CSS3

## 📖 How to Use
1. Enter the **Product Name** and **Quantity**.
2. Click **"Add Item"** to save it to the list.
3. Use the **"Delete"** button to remove items you no longer need.
